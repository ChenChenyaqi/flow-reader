# Quick Start

Get started with FlowReader in just a few minutes.

## Step 1: Configure Your API Provider

FlowReader uses a **BYOK (Bring Your Own Key)** model, giving you control over which AI service to use.

1. Select any text on a webpage
2. Click the üîç (magnifying glass) icon that appears
3. Click the **Settings** icon in the upper right corner of the FlowReader card
4. Go to the **Settings** tab
5. Choose your service provider:
   - **Zhipu AI (GLM-4.7)**: Recommended for users in China. [Get Free Key](https://open.bigmodel.cn/)
   - **OpenAI**: Standard GPT models. [Get API Key](https://platform.openai.com/api-keys)
   - **Custom**: Compatible with any OpenAI-format API (e.g., DeepSeek, Local LLM)
6. Enter your **API Key** and **Model Name**
7. Click **Save**

## Step 2: Set Your Vocabulary Level (Optional)

1. In the Settings card, switch to the **Vocabulary** tab
2. Select your current vocabulary level:
   - **500 words** (A1 - Beginner)
   - **1000 words** (A2 - Elementary)
   - **2000 words** (B1 - Intermediate) - Default
   - **3000 words** (B2 - Upper Intermediate)
   - **5000 words** (C1 - Advanced)
   - **8000 words** (C2 - Proficient)

This helps FlowReader filter vocabulary based on your proficiency.

## Step 3: Start Learning!

1. **Select Text**: Highlight any English sentence or paragraph
2. **Click the Icon**: Click the FlowReader icon that appears
3. **Explore**:
   - Read the **simplified version** streaming in
   - Check the **grammar highlights** (subject, predicate, object)
   - Learn **new vocabulary** with definitions and translations
   - Mark words as **"Known"** to filter them out in future analyses

## Tips for Best Results

- Start with shorter sentences (1-3 sentences) for clearer analysis
- Use the grammar visualization to understand sentence structures
- Regularly mark words you know to improve vocabulary filtering
- Try different AI models to find the best balance of speed and quality
