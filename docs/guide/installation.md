# Installation

FlowReader can be installed from browser stores or loaded as an unpacked extension for development.

## Option 1: Install from Web Store (Recommended)

FlowReader is available on:

- [Microsoft Edge Add-ons](https://microsoftedge.microsoft.com/addons/detail/fluentread/mnlilgfbjkbbimcjhneefdginlfefbal)

> Coming soon to Chrome Web Store

## Option 2: Load Unpacked (For Developers)

### Prerequisites

- Chrome, Edge, or any Chromium-based browser
- The FlowReader extension files

### Steps

1. **Download FlowReader**
   - Download the latest `flow-reader` zip from [GitHub Releases](https://github.com/ChenChenyaqi/flow-reader/releases)
   - Or build it yourself from source

2. **Open Extensions Page**
   - Chrome/Edge: Navigate to `chrome://extensions/`

3. **Enable Developer Mode**
   - Toggle the **"Developer mode"** switch in the top right corner

4. **Load the Extension**
   - Click the **"Load unpacked"** button
   - Select the `flow-reader` folder

5. **Refresh Your Browser**
   - After installation, refresh any open pages to enable FlowReader

### Verification

You should see the FlowReader icon in your extensions list. When you select text on any webpage, a magnifying glass icon should appear near your selection.
