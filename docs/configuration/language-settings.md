# Language Settings

FlowReader supports multiple interface languages for a personalized experience.

## Supported Languages

### English (en-US)

Full English interface for all menus, labels, and messages.

### 简体中文 (zh-CN)

Complete Chinese localization including:

- All UI elements
- Vocabulary translations
- Error messages
- Help text

## Changing Language

1. Open FlowReader Settings
2. Go to the **Vocabulary** tab
3. Find the **Language** section
4. Select your preferred language from the dropdown
5. The interface updates immediately - no refresh needed

## Language Persistence

Your language preference is saved locally and persists across:

- Browser sessions
- Page navigation
- Extension updates

## Translation Quality

We strive to provide accurate, natural translations. If you notice any issues:

- Report translation problems via [GitHub Issues](https://github.com/ChenChenyaqi/flow-reader/issues)
- Suggest improvements through pull requests

## Future Languages

We plan to add more languages in the future. Request your language by:

- Opening a GitHub issue
- Voting on existing language requests
