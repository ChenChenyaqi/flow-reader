# Configuration

Customize FlowReader to match your learning needs and preferences.

## Configuration Areas

- **API Setup** - Configure your AI provider and API key
- **Vocabulary Settings** - Set your vocabulary level and manage known words
- **Language Settings** - Choose your interface language

## Accessing Settings

1. Select any text on a webpage
2. Click the FlowReader icon that appears
3. Click the **Settings** icon in the upper right corner
