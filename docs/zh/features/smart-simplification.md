# 智能简化

FlowReader 的智能简化功能将复杂的英语文本转换为简单易懂的语言。

## 工作原理

当您选择文本时，FlowReader 使用 AI 来：

1. 分析所选文本的复杂性
2. 使用更简单的词汇（A1-A2 水平）重写它
3. 保留原始含义
4. 以打字机效果流式传输结果

## 简化内容

- 复杂的句子结构
- 高级词汇
- 技术术语
- 习语表达
- 冗长复杂的句子

## 示例

**原文：**

> "现代软件系统中复杂算法的实现需要对计算范式有全面的理解。"

**简化后：**

> "现代软件使用智能的计算机程序。您需要了解计算机的工作原理才能很好地使用它们。"

## 使用建议

- 每次选择 1-3 句以获得最佳效果
- 比较原文和简化版本以学习新模式
- 结合语法可视化与简化功能以深入理解
