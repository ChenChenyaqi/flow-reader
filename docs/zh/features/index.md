# 功能特性

FlowReader 提供强大的 AI 驱动工具，彻底改变您的阅读体验。

## 核心功能

### ⚡ 即时分析

在任意网页上选择文本，触发具有实时流式响应的即时 AI 分析。

### 📉 智能简化

使用打字机效果，将复杂的句子重写为简单易懂的语言（A1-A2 水平）。

### 🔍 语法可视化

自动高亮**主语**、**谓语**和**宾语**，帮助您快速理解句子结构。

### 📚 个性化词汇

- 根据您的 CEFR 等级（500~8000 词汇）过滤单词
- 提供简单的英语定义和中文翻译
- 记住您标记为"已认识"的单词
- 随时间追踪您的学习进度

### 🤖 多模型支持

原生支持：

- **智谱 AI (GLM-4.7)** - 速度快，适合中国用户
- **OpenAI** - 具有高级功能的 GPT 模型
- **自定义 API** - 任何兼容 OpenAI 的端点

### 🔒 隐私优先

您的数据保留在您的设备上：

- API 密钥本地存储
- 无服务器端数据收集
- 无分析或跟踪
- 纯前端应用
