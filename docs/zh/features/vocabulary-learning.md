# 词汇学习

FlowReader 帮助您根据熟练程度智能地构建词汇量。

## 智能过滤

FlowReader 根据您选择的等级过滤词汇：

| 等级 | 词汇量 | CEFR   | 描述               |
| ---- | ------ | ------ | ------------------ |
| 500  | A1     | 初级   | 基础日常词汇       |
| 1000 | A2     | 基础   | 常见会话词汇       |
| 2000 | B1     | 中级   | 标准词汇（默认）   |
| 3000 | B2     | 中高级 | 学术和专业词汇     |
| 5000 | C1     | 高级   | 复杂的词汇         |
| 8000 | C2     | 精通   | 接近母语水平的词汇 |

## 词汇信息

对于每个词汇单词，FlowReader 提供：

- **简单英语定义** - 易于理解的解释
- **中文翻译** - 母语翻译
- **上下文** - 在当前文本中的使用方式

## 标记单词

随着学习的进行，您可以将单词标记为：

- **已认识** - 您理解这个单词（将在未来被过滤）
- **不认识** - 您仍在学习中（将继续出现）
- **已忽略** - 您不想跟踪这个单词

## 进度跟踪

FlowReader 记住：

- 您标记为"已认识"的单词
- 随时间变化的词汇等级
- 跨会话的学习模式

所有词汇数据都存储在本地浏览器中。
