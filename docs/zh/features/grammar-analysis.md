# 语法分析

FlowReader 的语法可视化帮助您快速理解句子结构。

## 工作原理

分析文本时，FlowReader 识别并高亮三个关键组成部分：

### 主语（蓝色）

句子所涉及的人、地点、事物或想法。

### 谓语（绿色）

句子中的动作或存在状态。

### 宾语（橙色）

接受动词动作的名词或代词。

## 视觉显示

每个组件都带有彩色下划线高亮：

- <span style="border-bottom: 2px solid #3b82f6; padding-bottom: 2px;">主语</span> - 蓝色
- <span style="border-bottom: 2px solid #22c55e; padding-bottom: 2px;">谓语</span> - 绿色
- <span style="border-bottom: 2px solid #f97316; padding-bottom: 2px;">宾语</span> - 橙色

## 示例

**句子：**

> "The developer wrote code."

**分析：**

- **主语：** The developer
- **谓语：** wrote
- **宾语：** code

## 优势

- 快速识别句子结构
- 理解谁做了什么
- 学习常见的英语模式
- 改善您的写作
