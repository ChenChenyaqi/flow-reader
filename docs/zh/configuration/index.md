# 配置

自定义 FlowReader 以匹配您的学习需求和偏好。

## 配置区域

- **API 设置** - 配置您的 AI 提供商和 API 密钥
- **词汇设置** - 设置词汇等级并管理已认识的单词
- **语言设置** - 选择您的界面语言

## 访问设置

1. 在网页上选择任意文本
2. 点击出现的 FlowReader 图标
3. 点击右上角的 **设置** 图标
