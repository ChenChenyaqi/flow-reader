# 语言设置

FlowReader 支持多种界面语言，提供个性化体验。

## 支持的语言

### 英语 (en-US)

完整的英语界面，包括所有菜单、标签和消息。

### 简体中文 (zh-CN)

完整的中文本地化，包括：

- 所有 UI 元素
- 词汇翻译
- 错误消息
- 帮助文本

## 更改语言

1. 打开 FlowReader 设置
2. 进入 **词汇** 标签页
3. 找到 **语言** 部分
4. 从下拉菜单中选择您的首选语言
5. 界面立即更新 - 无需刷新

## 语言持久化

您的语言偏好将保存在本地，并持续存在于：

- 浏览器会话
- 页面导航
- 扩展更新

## 翻译质量

我们努力提供准确、自然的翻译。如果您发现任何问题：

- 通过 [GitHub Issues](https://github.com/ChenChenyaqi/flow-reader/issues) 报告翻译问题
- 通过拉取请求建议改进

## 未来语言

我们计划在未来添加更多语言。通过以下方式请求您的语言：

- 打开 GitHub issue
- 对现有的语言请求投票
