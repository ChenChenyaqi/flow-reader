# 词汇设置

根据您的熟练程度配置 FlowReader 如何处理词汇。

## 词汇等级

选择最适合您当前词汇量的等级：

### 500 词 (A1 - 初级)

适合刚刚开始学习英语的人。专注于最常见的单词。

### 1000 词 (A2 - 基础)

具有基本会话能力。包括日常词汇。

### 2000 词 (B1 - 中级) **[默认]**

适合舒适的日常阅读。涵盖大多数常见文本。

### 3000 词 (B2 - 中高级)

适合学术和专业阅读。包括更高级的词汇。

### 5000 词 (C1 - 高级)

适合复杂文本和细致理解。复杂的词汇。

### 8000 词 (C2 - 精通)

接近母语词汇水平。只有最先进的单词会被高亮。

## 设置等级

1. 打开 FlowReader 设置
2. 进入 **词汇** 标签页
3. 从下拉菜单中选择您的等级
4. 您的选择会自动保存

## 管理已认识的单词

随着单词的学习，您可以将它们标记为**"已认识"**以：

- 从未来的分析中过滤它们
- 专注于新词汇
- 追踪您的进度

### 批量操作

您可以：

- 一次将多个单词标记为"已认识"
- 导出已认识的单词列表
- 重置词汇历史

## 建议

- 如果您想看到更多词汇帮助，请从较低的等级开始
- 随着学习更多单词，向上调整
- 对您的水平要诚实 - 看到熟悉的单词作为提醒也是可以的
- 定期标记已认识的单词可提高过滤质量
