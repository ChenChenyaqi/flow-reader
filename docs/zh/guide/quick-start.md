# 快速上手

只需几分钟即可开始使用 FlowReader。

## 第一步：配置您的 API 提供商

FlowReader 采用 **BYOK（自备密钥）** 模式，让您完全控制使用哪个 AI 服务。

1. 在网页上选择任意文本
2. 点击出现的 🔍（放大镜）图标
3. 点击 FlowReader 卡片右上角的 **设置** 图标
4. 进入 **设置** 标签页
5. 选择您的服务提供商：
   - **智谱 AI (GLM-4.7)**：推荐中国用户使用 [获取免费密钥](https://open.bigmodel.cn/)
   - **OpenAI**：标准 GPT 模型 [获取 API 密钥](https://platform.openai.com/api-keys)
   - **自定义**：任何兼容 OpenAI 格式的 API（如 DeepSeek、本地 LLM）
6. 输入您的 **API 密钥** 和 **模型名称**
7. 点击 **保存**

## 第二步：设置词汇等级（可选）

1. 在设置卡片中，切换到 **词汇** 标签页
2. 选择您当前的词汇等级：
   - **500 词** (A1 - 初级)
   - **1000 词** (A2 - 基础)
   - **2000 词** (B1 - 中级) - 默认
   - **3000 词** (B2 - 中高级)
   - **5000 词** (C1 - 高级)
   - **8000 词** (C2 - 精通)

这有助于 FlowReader 根据您的熟练度过滤词汇。

## 第三步：开始学习！

1. **选择文本**：高亮选择任何英语句子或段落
2. **点击图标**：点击出现的 FlowReader 图标
3. **探索**：
   - 阅读**简化版本**的流式输出
   - 查看**语法高亮**（主语、谓语、宾语）
   - 学习**新词汇**及其定义和翻译
   - 将单词标记为**"已认识"**以在未来的分析中过滤它们

## 使用建议

- 首次使用选择较短的句子（1-3 句）以获得更清晰的分析
- 结合语法可视化与简化功能以深入理解
- 定期标记您认识的单词以改进词汇过滤
- 尝试不同的 AI 模型以找到速度和质量的最佳平衡
