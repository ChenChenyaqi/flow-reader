# 安装指南

FlowReader 可以从浏览器商店安装，或作为解压扩展加载用于开发。

## 方法一：从应用商店安装（推荐）

FlowReader 可在以下平台获取：

- [Microsoft Edge 加载项商店](https://microsoftedge.microsoft.com/addons/detail/fluentread/mnlilgfbjkbbimcjhneefdginlfefbal)

> 即将上架 Chrome 应用商店

## 方法二：加载解压包（适用于开发者）

### 前提条件

- Chrome、Edge 或任何基于 Chromium 的浏览器
- FlowReader 扩展文件

### 步骤

1. **下载 FlowReader**
   - 从 [GitHub Releases](https://github.com/ChenChenyaqi/flow-reader/releases) 下载最新的 `flow-reader` 压缩包
   - 或从源码自行构建

2. **打开扩展页面**
   - Chrome/Edge：导航至 `chrome://extensions/`

3. **启用开发者模式**
   - 切换右上角的 **"开发者模式"** 开关

4. **加载扩展**
   - 点击 **"加载已解压的扩展程序"** 按钮
   - 选择 `flow-reader` 文件夹

5. **刷新浏览器**
   - 安装后，刷新所有打开的页面以启用 FlowReader

### 验证

您应该在扩展列表中看到 FlowReader 图标。当您在网页上选择文本时，选区附近应该出现一个放大镜图标。
