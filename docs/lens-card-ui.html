Simple English

Because JavaScript doesn't wait for things to finish, it can cause "race conditions" (errors when timing is wrong) if you aren't careful.

Key Terms

Race Condition
A bug where the system's behavior depends on the uncontrollable timing of events.
Inevitable
Certain to happen; unavoidable.
