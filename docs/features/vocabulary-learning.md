# Vocabulary Learning

FlowReader helps you build vocabulary intelligently based on your proficiency level.

## Smart Filtering

FlowReader filters vocabulary based on your selected level:

| Level | Words | CEFR               | Description                     |
| ----- | ----- | ------------------ | ------------------------------- |
| 500   | A1    | Beginner           | Basic everyday words            |
| 1000  | A2    | Elementary         | Common conversational words     |
| 2000  | B1    | Intermediate       | Standard vocabulary (default)   |
| 3000  | B2    | Upper Intermediate | Academic and professional words |
| 5000  | C1    | Advanced           | Sophisticated vocabulary        |
| 8000  | C2    | Proficient         | Near-native vocabulary          |

## Word Information

For each vocabulary word, FlowReader provides:

- **Simple English Definition** - Easy-to-understand explanation
- **Chinese Translation** - Native language translation
- **Context** - How it's used in the current text

## Marking Words

As you learn, you can mark words as:

- **Known** - You understand this word (will be filtered in future)
- **Don't Know** - You're still learning (will continue to appear)
- **Ignored** - You don't want to track this word

## Progress Tracking

FlowReader remembers:

- Words you've marked as "Known"
- Your vocabulary level over time
- Learning patterns across sessions

All vocabulary data is stored locally in your browser.
